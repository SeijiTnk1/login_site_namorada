# 🔐 Login - Projeto Complementar

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como uma página de login personalizada para complementar o site principal criado em homenagem à minha namorada.

Seu objetivo é tornar a experiência mais divertida e imersiva antes do acesso ao conteúdo principal, funcionando como uma etapa inicial de entrada no site.

## 📸 Funcionalidades

* Tela de login personalizada
* Validação através de uma data especial
* Mensagens de erro para dados incorretos
* Redirecionamento para o site principal
* Design responsivo

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## ❤️ Observação

Este projeto foi criado apenas para complementar o site principal e deixar a experiência ainda mais especial.
